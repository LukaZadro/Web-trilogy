# UNIZG Net

<img src="frontend/public/logo_header.png" alt="Logo" width="180" height="180">

_**"Tvoja karijera počinje ovdje"**_

Web platforma koja povezuje **studente**, **alumni**, **poslodavce**, **udruge** i **sastavnice Sveučilišta u Zagrebu** na jednom mjestu.
Aplikacija za pomoć studentima pri pronalaženju studentskih praksi i poslova te spajanje s drugim studentima preko studentskih udruga.

## O projektu
Web platforma je izrađena u sklopu projektnog zadatka na natjecanju "Web trilogy 2025" u organizaciji studentske udruge **EESTEC LC Zagreb** (https://www.eestec.hr/).

Zadatak je obuhvaćao proces **od ideje do konačnog proizvoda** - ideja, dizajn, implementacija (front-end i back-end) i testiranje te konačno prezentaciju proizvoda.

## Tehnologije korištene u izradi
- **React** + **TypeScript** + **Tailwind** - front-end
- **Node.js** + **Express** - back-end
- **SQLite** - jednostavna baza podataka
